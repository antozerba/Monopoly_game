Per compilare il codice con CMake:
- Aprire la cartella con il progetto nel terminale
- Creare una cartella al suo interno tramite il comando
  mkdir build
- Navigare all'interno della cartella tramite il comando 
  cd build
- Per la scrittura dei file di cmake nella cartella /build digitare il comando 
  cmake ..
- Per la compilazione del codice digitare 
  make
- Per l'esecuzione del codice digitare 
  ./ProgettoFinale Human se vuole testare il gioco con il giocatore umano e 3 computer
  ./ProgettoFinale Computer se vuole testare il gioco con 4 computer come giocatori
